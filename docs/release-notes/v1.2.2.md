## Deutsch

### LAN-Kachel zeigt die Adresse zuverlässig an

- Ein Tipp auf die Schnelleinstellungs-Kachel startet den LAN-Empfang und zeigt
  die vollständige Sitzungsadresse in einem kompakten Fenster.
- Die Adresse ist auswählbar und lässt sich kopieren. Im selben Fenster kann
  der Empfang beendet werden.
- Läuft der Empfang bereits, öffnet ein weiterer Tipp die Adresse erneut.
- Der kurze sechsstellige Sitzungscode steht zusätzlich im Kachelnamen. Damit
  bleibt er auch auf Android-Oberflächen sichtbar, die den Untertitel einer
  Kachel nicht anzeigen.
- Der Ablauf unterstützt alle App-Versionen ab Android 13. Zufällige
  Sitzungsadresse, 30-Minuten-Limit und Android-17-Netzwerkfreigabe bleiben
  unverändert.

### Installation

`ThorROMButler-v1.2.2.apk` über die bestehende Installation installieren.
Einstellungen, eigene Systeme und Log bleiben erhalten.

## English

### LAN tile now shows the address reliably

- Tapping the Quick Settings tile starts LAN receive and displays the complete
  session address in a compact window.
- The address is selectable and can be copied. Receiving can be stopped from
  the same window.
- If receiving is already active, another tap shows the address again.
- The short six-character session code is also included in the tile label, so
  it remains visible on Android skins that hide tile subtitles.
- The flow supports every app version starting with Android 13. The random
  session address, 30-minute limit, and Android 17 network permission remain
  unchanged.

### Installation

Install `ThorROMButler-v1.2.2.apk` over the existing installation. Settings,
custom systems, and the action log are preserved.
