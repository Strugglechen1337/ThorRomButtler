## Deutsch

### Fortsetzbarer LAN-Empfang

- Die neue Browser-Warteschlange zeigt Fortschritt, Geschwindigkeit und
  Restzeit für jede Datei.
- Große Dateien werden in Blöcken übertragen. Abbrechen und Wiederholen sind
  direkt möglich; nach einer Unterbrechung wird dieselbe Datei bei erneuter
  Auswahl am letzten bestätigten Block fortgesetzt.
- Unvollständige Übertragungen bleiben unsichtbar. Erst die vollständig
  empfangene Datei wird atomar in den Download-Ordner übernommen; alte
  Teildateien werden nach sieben Tagen aufgeräumt.

### QR-Code und Verbindungsprüfung

- Die Empfangsansicht zeigt die geschützte Sitzungsadresse als QR-Code und
  bietet Kopieren sowie Teilen.
- Eine lokale Prüfung zeigt den Status von Netzwerkfreigabe, WLAN/Ethernet,
  LAN-Adresse, Download-Ordner, freiem Speicher und Empfangsserver.

### Exakte Duplikate und Diagnosebericht

- Nach der schnellen Bibliotheks-Prüfung kann optional eine bytegenaue
  SHA-256-Suche gestartet werden. Zur Beschleunigung werden nur gleich große
  Dateien gehasht und blockweise gelesen.
- Gefundene Gruppen und der theoretisch freigebbare Speicher werden angezeigt;
  Pfade und Prüfsummen landen im Sammlungsexport. Es wird nichts automatisch
  gelöscht.
- In den Einstellungen lässt sich freiwillig ein datenschutzfreundlicher
  Supportbericht teilen. Er enthält technische App-, Geräte-, Speicher-,
  Berechtigungs- und LAN-Daten, aber keine ROM-Namen oder Sitzungscodes.

### Installation

`ThorROMButler-v1.3.0.apk` über die bestehende Installation installieren.
Einstellungen, eigene Systeme und Log bleiben erhalten.

## English

### Resumable LAN receive

- The new browser queue shows progress, speed, and ETA for every file.
- Large files are transferred in chunks. Cancel and retry are available
  directly; after an interruption, selecting the same file continues from the
  last confirmed chunk.
- Incomplete transfers stay hidden. Only the fully received file is committed
  atomically to Downloads, and stale partial files are cleaned after seven
  days.

### QR code and connection checks

- The receive screen displays the protected session address as a QR code and
  provides copy and share actions.
- A local check reports network permission, Wi-Fi/Ethernet, LAN address,
  download folder, free space, and receive server reachability.

### Exact duplicates and diagnostic report

- After the fast library check, an optional byte-exact SHA-256 scan can be
  started. Only same-sized candidates are hashed and files are read in chunks.
- Duplicate groups and theoretically reclaimable storage are shown; paths and
  hashes are included in the collection export. Nothing is deleted
  automatically.
- Settings now provide a voluntarily shared, privacy-friendly support report.
  It contains technical app, device, storage, permission, and LAN details, but
  no ROM names or session codes.

### Installation

Install `ThorROMButler-v1.3.0.apk` over the existing installation. Settings,
custom systems, and the action log are preserved.
