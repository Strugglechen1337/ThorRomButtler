## 🇩🇪 Deutsch

### System-Packs und eigene Systeme

- **Eigene Systeme:** Zusätzliche Systeme lassen sich direkt in den
  Einstellungen anlegen, bearbeiten und entfernen.
- **JSON-System-Packs:** Eigene Definitionen können über
  `ThorRomButler-system-pack.json` im Download-Ordner importiert und exportiert
  werden. Das dokumentierte Schema v1 unterstützt Endungen, Confidence-Level,
  kontrollierte Magic-Regeln und optionale Spiel-Unterordner.
- **Sichere Vorschau:** Vor der Installation zeigt die App Pack-Name,
  Systemanzahl und alle mehrdeutigen Endungen. Konflikte bleiben `UNKNOWN` und
  werden niemals automatisch zugeordnet.
- Eingebaute Systeme und ihre Zielordner können von einem eigenen Pack nicht
  überschrieben werden. Packs werden ausschließlich lokal und niemals
  automatisch aus dem Internet geladen.

### Sicherung und Qualität

- Einstellungs-Backups werden zuerst vollständig geprüft und anschließend in
  einer atomaren DataStore-Transaktion übernommen. Ungültige oder unsichere
  Zielordner werden abgelehnt, ohne Einstellungen teilweise zu verändern.
- Die Editor- und Importregeln verwenden dieselbe zentrale Validierung.
- 101 Unit-Tests, Android Lint und der Build der Compose-Test-APK laufen vor dem
  Release. GitHub-Beschreibung, Issue-Vorlagen und Dokumentation sind Deutsch
  und Englisch gepflegt.

### Installation

`ThorROMButler-v1.2.0.apk` über die bestehende Installation installieren —
Einstellungen, eigene Systeme und Log bleiben erhalten.

## 🇬🇧 English

### System packs and custom systems

- **Custom systems:** Additional systems can be created, edited, and removed
  directly in Settings.
- **JSON system packs:** Custom definitions can be imported and exported as
  `ThorRomButler-system-pack.json` in the download folder. The documented v1
  schema supports extensions, confidence levels, controlled magic rules, and
  optional per-game folders.
- **Safe preview:** Before installation, the app shows the pack name, system
  count, and every ambiguous extension. Conflicts remain `UNKNOWN` and are
  never assigned automatically.
- Custom packs cannot overwrite built-in systems or their target folders.
  Packs are local only and are never downloaded automatically.

### Backup and quality

- Settings backups are validated completely before being applied in one atomic
  DataStore transaction. Invalid or unsafe target folders are rejected without
  partially changing settings.
- The editor and importer use the same central validation rules.
- 101 unit tests, Android Lint, and the Compose test APK build run before the
  release. The GitHub description, issue forms, and documentation are
  maintained in German and English.

### Installation

Install `ThorROMButler-v1.2.0.apk` over the existing installation — settings,
custom systems, and the action log are preserved.
