## 🇩🇪 Deutsch

### Neu

- **🩹 ROM-Patcher (IPS/UPS/BPS)**: Fan-Übersetzungen und Romhacks direkt auf dem Gerät anwenden. Einfach die Patch-Datei mit demselben Dateinamen wie die ROM in den Download-Ordner legen (z. B. `Spiel.sfc` + `Spiel.bps`) — der Scan zeigt das Paar, ein Tipp erzeugt das gepatchte Spiel als neue Datei. Original und Patch bleiben unangetastet; bei UPS/BPS wird per Prüfsumme verifiziert, dass die ROM wirklich zur Patch-Basis passt.
- **💿 M3U-Playlists für Multi-Disc-Spiele**: Landen mehrere Discs desselben Spiels in einem Systemordner, legt der Butler automatisch die passende `.m3u` an — ein Eintrag im Frontend, funktionierender Disc-Wechsel im Emulator. Vorhandene Playlists werden nie überschrieben; abschaltbar in den Einstellungen.
- **✍️ Auf DAT-Namen umbenennen** (Opt-in): Verifizierte Einzeldatei-ROMs werden beim Einsortieren auf ihren offiziellen No-Intro/Redump-Namen umbenannt — aufgeräumte Bibliothek ohne Handarbeit. Multi-File-Spiele (`.bin`+`.cue`) werden nie angefasst.
- **🎛️ Frontend-Profile**: Einstellungen → „Frontend-Profil" stellt alle Systemordner in einem Schritt auf die Konvention von **Batocera/Knulli** oder **Onion OS (Miyoo)** um. ES-DE bleibt der Standard; einzelne Ordner lassen sich weiterhin anpassen.

### Installation

`ThorROMButler-v1.4.0.apk` über die bestehende Installation installieren — Einstellungen bleiben erhalten.

## 🇬🇧 English

### New

- **🩹 ROM patcher (IPS/UPS/BPS)**: apply fan translations and romhacks directly on the device. Just drop the patch file with the same file name as the ROM into the download folder (e.g. `Game.sfc` + `Game.bps`) — the scan shows the pair, one tap creates the patched game as a new file. Original and patch stay untouched; for UPS/BPS, checksums verify that the ROM really matches the patch base.
- **💿 M3U playlists for multi-disc games**: when several discs of the same game land in a system folder, the butler automatically writes the matching `.m3u` — one frontend entry, working disc swap in the emulator. Existing playlists are never overwritten; can be disabled in Settings.
- **✍️ Rename to DAT names** (opt-in): verified single-file ROMs are renamed to their official No-Intro/Redump name when sorted in — a tidy library without manual work. Multi-file games (`.bin`+`.cue`) are never touched.
- **🎛️ Frontend profiles**: Settings → "Frontend profile" switches all system folders to the **Batocera/Knulli** or **Onion OS (Miyoo)** convention in one step. ES-DE remains the default; individual folders can still be adjusted.

### Install

Install `ThorROMButler-v1.4.0.apk` over the existing app — settings are kept.
