## 🇩🇪 Deutsch

### Sicheres Einsortieren

- **Atomisches Ersetzen:** Neue ROM-Dateien werden zunächst vollständig in eine
  versteckte Teildatei geschrieben und geprüft. Eine vorhandene ROM bleibt bis
  zum erfolgreichen Abschluss gesichert und wird bei einem Fehler automatisch
  wiederhergestellt.
- Abgebrochene Vorgänge hinterlassen keine sichtbaren halbfertigen ROMs. Nach
  einem unerwarteten Prozessende kann die App zurückgebliebene Sicherungen beim
  nächsten Einsortieren wiederherstellen.
- Quellarchive bleiben bei neuen beziehungsweise bisher unveränderten
  Einstellungen jetzt standardmäßig erhalten. Löschen und der siebentägige
  Papierkorb bleiben optional verfügbar.
- **Rückgängig** erkennt ein Quellarchiv nun auch im Papierkorb.

### Sicherer Dateiempfang

- Der LAN-Empfang verwendet pro Start eine zufällige, einmalige Sitzungsadresse
  und beendet sich automatisch nach 30 Minuten.
- LAN-Uploads und über Android geteilte Dateien werden unter bereinigten
  Dateinamen zunächst vollständig geschrieben und erst danach für den Scanner
  sichtbar. Der LAN-Empfang prüft außerdem den freien Speicherplatz.
- Android-15-Timeouts der Hintergrunddienste werden sauber behandelt.

### Qualität

- Die zuvor vorhandenen Android-Lint-Fehler wurden behoben. Unit-Tests und Lint
  laufen jetzt sowohl in der normalen CI als auch vor jedem Release.

### Installation

`ThorROMButler-v1.1.1.apk` über die bestehende Installation installieren —
Einstellungen und Log bleiben erhalten.

## 🇬🇧 English

### Safer sorting

- **Atomic replacement:** New ROM files are first written completely to a
  hidden partial file and verified. An existing ROM stays backed up until the
  operation succeeds and is restored automatically after a failure.
- Cancelled operations never expose half-written ROMs. If the process ends
  unexpectedly, a leftover backup can be recovered during the next sorting run.
- Source archives are now kept by default for new or previously unchanged
  settings. Immediate deletion and the seven-day trash remain optional.
- **Undo** now also recognizes a source archive inside the trash folder.

### Safer file receiving

- LAN receive uses a random one-time session address on every start and stops
  automatically after 30 minutes.
- LAN uploads and files shared through Android use sanitized names, are written
  completely before becoming visible to the scanner, and never silently
  overwrite an existing file. LAN receive also checks available storage.
- Android 15 foreground-service timeouts are handled gracefully.

### Quality

- The existing Android Lint errors have been fixed. Unit tests and Lint now run
  in both regular CI and every release build.

### Installation

Install `ThorROMButler-v1.1.1.apk` over the existing installation — settings and
the action log are preserved.
