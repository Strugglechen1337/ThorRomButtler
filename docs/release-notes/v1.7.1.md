## 🇩🇪 Deutsch

### Verbessert

- **🔍 Der BIOS-Ordner schlägt sich selbst vor**: Die App durchsucht die üblichen Orte (`BIOS/`-Ordner auf allen Speichern inkl. SD-Karte, `RetroArch/system`, `bios/` neben dem ROM-Ordner) und bevorzugt Ordner, die **bereits bekannte BIOS-Dateien enthalten** — das stärkste Signal, dass es der richtige ist. Ein Tipp auf „Übernehmen" genügt. Existiert noch nichts, schlägt die App vor, einen `BIOS`-Ordner neben dem ROM-Ordner anzulegen.
- **🐬 Core-Konventionen automatisch**: Dreamcast-BIOS (`dc_boot.bin`, `dc_flash.bin`, `naomi.bin`) landet beim Einsortieren jetzt automatisch im Unterordner `dc/`, wie Flycast es erwartet. Alles andere bleibt direkt im BIOS-Ordner.

Hintergrund: RetroArchs konfiguriertes System-Verzeichnis liegt in `Android/data` und ist für andere Apps seit Android 13 unlesbar — mehr „selbst herausfinden" als diese Dateisystem-Heuristik ist auf Android technisch nicht möglich. Einmalig RetroArchs System-Verzeichnis auf den gewählten Ordner umstellen, dann passt alles zusammen.

### Installation

`ThorROMButler-v1.7.1.apk` über die bestehende Installation installieren — Einstellungen bleiben erhalten.

## 🇬🇧 English

### Improved

- **🔍 The BIOS folder now suggests itself**: the app checks the usual places (`BIOS/` folders on all volumes incl. SD card, `RetroArch/system`, `bios/` next to the ROM folder) and prefers folders that **already contain known BIOS files** — the strongest signal that it's the right one. One tap on "Apply" is enough. If nothing exists yet, the app proposes creating a `BIOS` folder next to your ROM folder.
- **🐬 Core conventions handled automatically**: Dreamcast BIOS (`dc_boot.bin`, `dc_flash.bin`, `naomi.bin`) now automatically lands in the `dc/` subfolder as expected by Flycast. Everything else stays directly in the BIOS folder.

Background: RetroArch's configured system directory lives in `Android/data`, which is unreadable for other apps since Android 13 — more "figuring it out" than this file-system heuristic is technically impossible on Android. Point RetroArch's system directory at the chosen folder once and everything lines up.

### Install

Install `ThorROMButler-v1.7.1.apk` over the existing app — settings are kept.
