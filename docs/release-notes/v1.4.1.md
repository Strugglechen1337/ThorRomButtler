## Deutsch

### 7z-Archive werden wieder korrekt entpackt

- Ein lesbares Archiv ohne automatisch erkannte ROM-Endung wurde bisher nach
  einer manuellen Systemwahl fälschlich als komplette Datei in den Zielordner
  verschoben. Das betraf besonders 7z-Dateien mit unbekannten oder eigenen
  Dateiendungen im Inneren.
- Diese Inhalte bleiben jetzt als echte Archiveinträge erhalten und werden
  nach der manuellen Zuordnung regulär entpackt. Das gilt für 7z, RAR4 und ZIP.
- Nur eine klar begrenzte Ausnahme bleibt gepackt: Ein ZIP, das ausdrücklich
  Arcade oder Neo Geo zugeordnet wurde, wird als vollständiges ROM-Set
  verschoben.
- Die Zuordnungsansicht benennt jetzt sichtbar, ob unbekannte Archivinhalte
  entpackt werden und wann die Arcade-Ausnahme gilt.

### Installation

`ThorROMButler-v1.4.1.apk` über die bestehende Installation installieren.
Einstellungen, eigene Systeme und Log bleiben erhalten.

## English

### 7z archives are extracted correctly again

- A readable archive without an automatically recognized ROM extension was
  previously moved as a complete file after manual system assignment. This
  especially affected 7z files containing unknown or custom extensions.
- Those contents now remain real archive entries and are extracted normally
  after manual assignment. This applies to 7z, RAR4, and ZIP.
- Only one deliberately narrow exception stays packed: a ZIP explicitly
  assigned to Arcade or Neo Geo is moved as a complete ROM set.
- The assignment view now clearly states that unknown archive contents are
  extracted and when the Arcade exception applies.

### Installation

Install `ThorROMButler-v1.4.1.apk` over the existing installation. Settings,
custom systems, and the action log are preserved.
